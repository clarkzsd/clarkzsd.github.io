  4. The pits I met when I use MongoDB

    2017-04-23
    在服务器上装了 MongDB 后，想在本地环境用可视化工具来管理 MongoDB ，可是死活连不上。后来发现在配置文件`/etc/mongod.conf`中，其默认的 bindIp 为 `127.0.0.1`，也就是说数据库只限内网访问。将其改为`0.0.0.0`或其他指定环境即可。若想指定多个 ip ，每个 ip 之间用逗号分隔。
