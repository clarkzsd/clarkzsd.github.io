我大概也是个无趣的人吧

2017-05-19

我常常觉得自己遇到的人没有以前遇到的有趣了。

学校里大家各自干自己的事，互相不甚了解，若聊上几句也是十分默契地马上打住，用现在的流行语来讲就是“尬聊”吧。

我经常觉得诧异为什么很多人对世界上那些有趣的事物一无所知呢？他们平常就没有特别热爱某些事吗？而且很奇怪他们竟然也生活的挺开心的。他们不知道自己很无趣吗？但仔细一想，他们也有能使自己愉悦的兴趣爱好，他们的精神世界或许很富足。只是我无法理解罢了。

2017-04-26

网络基础 TCP/IP

在讨论 Socket 之前，我们得先追本溯源，看看 TCP/IP 是什么。

计算机与网络设备之间的通信，就如大国之间的外交一样，需要一种规则。这种规则就成称为协议（protocol）。计算机网络协议中存在各种各样的内容：从电缆规格到 IP 地址的选定方法、双方建立通信的顺序，以及 Web 页面显示需要处理的步骤，等等。
